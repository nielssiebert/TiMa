# TiMa

TiMa is a timing and automation platform for managing scheduled actions, grouped executions, triggers, sequences, and runtime factors through a Vue frontend and a Flask backend. It is built for operating time-based workflows with a clear admin UI, an API-driven backend, and a deployment path that also fits Raspberry Pi installations.
